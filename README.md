# MPHSI School Portal

A React + Vite school portal featuring Home, About, Gallery, Contact, and Login pages with dark mode, announcements calendar, gallery carousel, and responsive navigation.

## Prerequisites
- Node.js 18+ and npm

## Installation
```bash
npm install
```

## Development
```bash
npm run dev
```
Open the printed local URL in your browser.

## Production Build
```bash
npm run build
npm run preview
```

## Project Notes
- Dark mode toggle with minimalist sun/moon icon
- Announcements & events with calendar filtering and month/year selectors
- Gallery with per-category carousels and thumbnail navigation
- Mobile-friendly navigation (burger menu) and back-to-top button
- The favicon and navbar logo use `public/school-logo.png`
